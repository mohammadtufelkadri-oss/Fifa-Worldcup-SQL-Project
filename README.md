# Fifa-Worldcup-SQL-Project #
This project focuses on analyzing FIFA dataset using SQL by performing various data exploration and analytical queries. It includes operations on multiple tables such as teams, players, and totalgroups to extract meaningful insights about team performance and group statistics.

The project covers key SQL concepts including:

Data filtering and sorting
Aggregations (COUNT, AVG, SUM, MAX, MIN)
Joins between tables
Group-wise analysis
Calculated metrics like win percentage, loss percentage, and goal differences

Through these queries, insights such as team rankings, match performance, scoring efficiency, and group-level comparisons are derived. The project demonstrates practical use of SQL for sports data analysis and helps in building strong fundamentals in querying and data analysis.
